При обработке добавления каналов всегда проверяй два сценария: 1) Бот добавляется в канал (my_chat_member events) 2) Бот уже в канале, но пользователь использует кнопку добавления (?startchannel параметры в /start). Для второго случая добавляй обработку в start handler с проверкой прав пользователя и статуса бота.
