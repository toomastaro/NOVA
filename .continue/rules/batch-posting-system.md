Папки используются ТОЛЬКО для каналов (FolderType.CHANNEL). Никогда не создавать FolderType.BOT. Все выбранные каналы в одной операции образуют одну batch-отправку с одинаковым контентом. Редактирование или удаление через бота применяется СРАЗУ ко всем каналам этой batch-отправки.
